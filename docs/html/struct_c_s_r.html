Detailed Description

Compressed Sparse Row matrix.

Data format same as of scipy.sparse.csr_matrix. Is the standard CSR representation where the column indices for row i are stored in indices[indptr[i]:indptr[i+1]] and their corresponding values are stored in data[indptr[i]:indptr[i+1]].
