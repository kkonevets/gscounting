Detailed Description

template<class T>
class ExternalSorter< T >

Sorts file on disk using merge sort.

First it splits file on parts, then sorts them (consuming max_mem at a time), saves parts on disk to save_dir and then merges those parts while lazy loading. Uses priority queue for merging (memory consumption is minimal).

Parameters
save_dirName of a directory to save parts in
max_memMaximum size of a part file in bytes
