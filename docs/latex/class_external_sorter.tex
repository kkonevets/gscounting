\subsubsection*{template$<$class T$>$\newline
class External\+Sorter$<$ T $>$}

Sorts file on disk using merge sort. 

First it splits file on parts, then sorts them (consuming {\ttfamily max\+\_\+mem} at a time), saves parts on disk to {\ttfamily save\+\_\+dir} and then merges those parts while lazy loading. Uses priority queue for merging (memory consumption is minimal).
